แบบประเมินความเครียด (ST-5)

แบบประเมินความเครียด (ST-5) นี้ จัดทำขึ้นเพื่อประเมินระดับความเครียดของผู้ตอบ ในช่วง 2–4 สัปดาห์ที่ผ่านมา โดยให้เลือกคำตอบที่ตรงกับความรู้สึกหรืออาการของตนเองมากที่สุด

0 คะแนน = เป็นน้อยมากหรือแทบไม่มี 1 คะแนน = เป็นบางครั้ง 2 คะแนน = เป็นบ่อยครั้ง 3 คะแนน = เป็นประจำ
การแปลผล
🟢 0–4 คะแนน = เครียดน้อย
🟡 5–7 คะแนน = เครียดปานกลาง
🟠 8–9 คะแนน = เครียดมาก
🔴 10–15 คะแนน = เครียดมากที่สุด
